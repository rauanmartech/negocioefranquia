# Projeto

Cliente: Rodrigo

Projeto: Reformulação da área de notícias do portal Negócios e Franquias.

## Site Atual

URL:
https://negocioefranquia.com.br/

## Objetivo

Substituir completamente a experiência atual da área de notícias.

O WordPress continuará sendo utilizado como CMS.

Toda a publicação de conteúdo continuará sendo feita pelo WordPress.

O novo frontend deverá consumir os dados através da REST API já existente.

## Tecnologias

Frontend:
- Next.js
- Deploy Vercel

Backend:
- WordPress REST API

## APIs disponíveis

/wp-json/wp/v2/posts
/wp-json/wp/v2/categories
/wp-json/wp/v2/media
